### Meta-Agentics
*Definition:* The discipline of designing agents whose primary function is to create, configure, or improve other agents.
*   **Meta-Prompting**: Writing prompts that generate other prompts.
*   **The "Expertise File"**: A dedicated memory file (YAML/MD) where an agent records its learned lessons and "mental model" of a domain, distinct from the codebase.

### Active Working Memory
*Definition:* Systems that maintain context across sessions not just through vector stores, but through structured, editable documents (Semantic Memory).
